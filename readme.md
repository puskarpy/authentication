## Authentication system
Full-stack authentication system built with React, Node.js, Express, and MongoDB featuring JWT-based auth, HTTP-only cookies, persistent login, protected routes, and secure token handling using React Query.

### Features

- User Signup & Login with form validation

- JWT-based authentication (Access & Refresh tokens)

- HTTP-only cookies for secure refresh token storage

- Bearer token authorization for protected API routes

- Persistent login (session survives page refresh)

- Protected routes on both frontend and backend

- Secure logout & token invalidation

- Error handling and standardized API responses

### Tech
- **Frontend**: React, React Query (TanStack Query), Axios

- **Backend**: Node.js, Express.js

- **Database**: MongoDB, Mongoose

- **Auth & Security**: JWT, Cookies, Middleware
